# Vue Todo List

## how to start

- npm i
- npm run serve

## What are used

- [x] vue
- [x] fontawesome
- [x] google font
- [x] localstorage to save todo items
- [x] vue Modal when none in input
- [x] vue list Transition
- [x] vuex (state, mutations, commit)
- [x] Component binding helpers (mapGetters, mapMutations)
